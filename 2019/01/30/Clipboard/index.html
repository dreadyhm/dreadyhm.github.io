Vue中实现复制粘贴

发表于 2019-01-30 | 分类于 前端

背景

最近的一个项目要实现类似淘口令的功能，而在这个功能中复制粘贴是很重要的一环，那复制粘贴是如何实现的呢？

步骤

1.使用Clipboard.js，npm引入

npm install clipboard --save

2.在组件中引入

import ClipboardJS from 'clipboard';

3.html，需要拷贝的内容是input框中的内容

4.js，点击拷贝按钮，触发copy事件，在success和error事件中各自定义复制成功或者失败的逻辑。

注意点

1.在实现功能的时候，出现了需要点击两下，才能复制成功，后来发现，自己在使用的过程中，参考网上的攻略，新建对象时， 是new Clipboard，而官网(https://clipboardjs.com/)是`new ClipboardJS`，修改之后，就能一次成功复制了。

2.在安卓机型中，点击拷贝，input若不设置为只读，会调起手机键盘，会给安卓用户带来一些不便利。因此，最好将input设置为只读。
