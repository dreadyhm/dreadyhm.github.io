实验三 使用Github管理Android项目

一、实验目的

  • 掌握Android Studio中Git使用
  • 掌握Github管理项目

二、实验任务

  • 在Android Studio中新建项目
  • 添加VCS中的Git支持
  • 发布项目到Github
  • 在Android Studio中进行修改代码、提交、新建分支、合并等操作
  • 推送项目到Github
  • 如有多人合作，处理pull request

三、实验步骤与结果

  • 在Android Studio中新建Android工程
  • 在Android Studio中，选择VCS 下的import intoVersion Control选项，ShareProject on Github
  • 登陆Github账号
  • 点击Share，将Android工程分享到Github上
  • 点击OK，将Android工程中的所有文件分享到Github上
  • 对于 Add File to Git 提示框，选择Yes
  • Android工程成功推送后，Android Studio的右上角会出现“Successfully shared projecton GitHub”的提示
  • 在github上，可看到已成功推送的工程，如下图：

  • 修改Android工程的代码，右键选择Git commit file，将修改过的工程commit and push到Github上
  • 由下图可知，Android工程已修改过

四、实验小结

  • 通过本实验，我学会了将Android工程推送到github上，这样可以方便的将我们的工程上传到网上，保存下来。如果我们的身边没有代码，也可以在github上下载自己的工程。
