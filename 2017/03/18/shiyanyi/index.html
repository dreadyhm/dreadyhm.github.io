实验一 在Github上用Markdown创建技术博客

一、实验目的

  1. 熟悉Github网站

  2. 掌握Markdown语法

二、实验任务

  1. 注册Github账号

  2. 创建Gihub Pages repository

  3. 发布第一篇博客

  4. 学习Markdown

  5. 利用Hexo生成博客并部署到Github

三、实验步骤与结果

1.注册账号，我的用户名是dreadyhm

2.我创建GihubPages repository的是the-first

3.pull request

​ 4.

  • 安装git
  • 安装node
  • 设置npm镜像为淘宝:npm config set registry https://registry.npm.taobao.org
  • 安装hexonpminstall -g hexo-cli
  • 初始化站点hexo initmyhexo
  • cd hexo
  • npm install
  • 浏览本地站点:hexoserver
  • 在Github上新建repository，dreadyhm.github.io

  • 修改配置文件_config.yml的deploy部分

    ​ type: git

    ​ repository:git@github.com:dreadyhm/dreadyhm.github.io.git

    ​ branch: master

  • 安装hexo-deployer-git

    ​ npm install hexo-deployer-git –save

  • 配置git全局邮箱与名字

    ​ git config –global user.email”865961680@qq.com”

    ​ git config –global user.name “dreadyhm”

  • 生成博客hexo generate

  • 部署博客hexodeploy

5.将博客部署到github上

四、实验小结

  • 通过本实验，学会了在github上部署hexo，这个过程会出现很多的问题，但是只要细心，不漏掉步骤，就可以轻松的完成博客。
