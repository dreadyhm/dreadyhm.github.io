Vue 引入阿里巴巴矢量图标库图标

发表于 2018-04-23 | 分类于 前端 |

背景

在校园智能柜项目中，使用了Element UI框架，但是在Element自身提供的icon图标数较少，且与项目所需图标的匹配度不高，故需要使用第三方图标库，此次使用的是阿里巴巴矢量图标库。

使用步骤

1.打开阿里巴巴矢量图标库（地址：iconfont.cn）。

2.进入网站登录账户之后，搜索选择想要的图标，点击购物车按钮，添加到购物车。

3.将图标添加至新建项目

  • 选择更多操作->编辑项目
  • 编辑项目名称：（如：my）
  • 修改前缀，为避免与el-icon的自带图标发生冲突，一般在el-icon后+项目名称，（如：el-icon-my）
  • 设置Font Family后，保存。

4.下载至本地。

5.在项目中，/src/assets/目录下新建icon文件夹，将下载的文件放至此文件夹中。
